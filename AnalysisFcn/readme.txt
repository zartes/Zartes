Preinstalación del software ZarTES Analysis.

- Sin MATLAB instalado:

    Es necesario antes de ejecutar el programa, instalar el MCRInstaller del 2017.
    La carpeta donde se encuentra el archivo está en Google Drive en:
    X-IFU/Software/MCR2017/MCR_R2017a_win64_installer.exe

- Con MATLAB instalado:
    
    Si la versión de MATLAB instalada no corresponde con la versión de MATLAB R2017a,
    el usuario deberá realizar el punto anterior. 

Instalación del Software de ZarTES Analysis

- El archivo Analysis.exe debe de ir acompañado por el archivo TES_Analyzer_UserGuide.pdf
  para su correcto funcionamiento. 

Reporte de errores
 
- Los posibles errores o mal funcionamiento del software debe ser comunicado al correo
  jbolea@unizar.es identificando el workflow que ha llevado a ese error o mal funcionamiento.
